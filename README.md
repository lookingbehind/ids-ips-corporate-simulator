# Симулятор атаки на корпоративную сеть с IDS/IPS

## 📋 Описание проекта

Это учебный проект, разработанный для курсовой работы по теме **"Развёртывание системы обнаружения и предотвращения вторжений в корпоративной сети"**.

Проект представляет собой симулятор атаки на компьютеры корпоративной сети, защищённой системой обнаружения и предотвращения вторжений (IDS/IPS). В качестве IDS используется **ViPNet IDS NS VA**, а атаки выполняются с помощью **Kali Linux** на целевые системы **Windows**.

## 🎯 Цель проекта

Демонстрация работы систем IDS/IPS в реальных условиях:
- Обнаружение сетевых атак
- Анализ трафика с помощью DPI (Deep Packet Inspection)
- Мониторинг и регистрация инцидентов безопасности
- Интеграция с SIEM-системами

## 🏗️ Архитектура системы

### Сетевая топология

Симулятор использует три изолированные сети:

- **DMZ** (192.168.10.0/24) — зона атакующего (Kali Linux)
- **LAN** (192.168.20.0/24) — внутренняя корпоративная сеть (Windows)
- **Management** (192.168.30.0/24) — сеть управления IDS

### Компоненты

1. **Kali Linux** — машина атакующего
2. **Windows 10/11** — целевая система
3. **ViPNet IDS NS VA** — система обнаружения вторжений
4. **VMware Workstation/VirtualBox** — платформа виртуализации

## 📦 Требования

### Аппаратные требования

- **Процессор**: 4+ ядра с поддержкой виртуализации (Intel VT-x / AMD-V)
- **Оперативная память**: 16+ ГБ
- **Дисковое пространство**: 100+ ГБ свободного места

### Программное обеспечение

- VMware Workstation Pro/Player 16+ или VirtualBox 6+
- Kali Linux (последняя версия)
- Windows 10/11 или Windows Server
- ViPNet IDS NS VA (OVA-файл)

## 🎭 Сценарии атак

Проект включает следующие типы атак:

### 1. Разведка (Reconnaissance)
- Сканирование портов (Nmap)
- Определение версий служб
- Сканирование уязвимостей

### 2. Поиск уязвимостей
- Веб-сканирование (Nikto)
- Проверка SMB-уязвимостей
- Поиск EternalBlue (MS17-010)

### 3. Эксплуатация (Exploitation)
- Использование Metasploit Framework
- Эксплуатация MS17-010
- Получение Meterpreter-сессии

Подробнее: [attacks/README.md](attacks/README.md)

## 📊 Мониторинг и анализ

### Обнаруживаемые события

- ⚠️ **INFO**: Сканирование сети (Network Scanning)
- ⚠️ **WARNING**: Разведка уязвимостей (SMB Enumeration)
- 🔴 **CRITICAL**: Эксплуатация уязвимостей (Exploitation)
- 🔴 **CRITICAL**: Компрометация системы (Meterpreter Session)

### Интеграция с SIEM

ViPNet IDS поддерживает экспорт событий через Syslog в форматах:
- CEF (Common Event Format)
- Syslog RFC 5424

Подробнее: [docs/siem-integration.md](docs/siem-integration.md)

## 📈 Результаты

После проведения атак IDS регистрирует:

- **True Positives (TP)** — истинно положительные срабатывания
- **False Positives (FP)** — ложные срабатывания
- **False Negatives (FN)** — пропущенные атаки

Метрики эффективности:
- **Precision** = TP / (TP + FP)
- **Recall** = TP / (TP + FN)
